The Idea

I played Temtem a long time ago, when it was in beta and I had a blast. It was a challenging game. The combat was improved upon its rival Pokemon, adding more layers of complexity, and the overall MMORPG was an enhancing experience lacking in other genre-like games.

That's why I decided to build a site where you can share your experiences, look up different temtem information to guide you in your journey and ultimately use the page as a tool to enhance the gaming experience.

Features

  • A social media type page where you can share everything about the temtem universe, your opinions about the features of the game and discussions related to battle strategies.

  • Information about every Temtem, including their traits, techniques, and different stats, presented in a delightful UI.

  • The classic type chart, which lets you check your temtem weaknesses and strengths and a technique effectiveness calculator.

The Stack

The app is built using Python and Flask, which uses sqlite3 and consequently SQL to query all the data about the users and the posts. It connects with a fan-made api fetching a bunch of data about the temtem universe. Lastly, it uses Jinja for the templating and a Bootstrap theme for the styling.
